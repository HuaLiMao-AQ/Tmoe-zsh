#### igorsilva

![igorsilva theme](https://raw.githubusercontent.com/igor9silva/zsh-theme/master/igorsilva.gif)

**What it shows**

- The current folder
- A customizable delimiter
- The current branch
- The current branch state (shows `*` if has uncommited changes)
